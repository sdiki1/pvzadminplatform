WB Wildberries — правила начисления
  • База: персональная ставка сотрудника; если смена < 8 ч — почасовая ставка × часы
  • Приёмка: сумма в ₽ из статистики, привязывается к смене
  • Раздел смены: несколько менеджеров на одной точке в один день — приёмка и выданные делятся пропорционально часам
  • Бонус за выдачу: за каждые полные {{ wb_issue_bonus_step }} шт. → +{{ wb_issue_bonus_amount }} ₽
    Пример: 121 шт → +100 ₽  |  199 шт → +100 ₽  |  200 шт → +200 ₽
OZON Ozon — правило

1 смена = 1 900 ₽ фиксированно.
Если у сотрудника задана ставка за смену — используется она.

Бонусы менеджера (выплата 10-го)
  • Тип 1: +{{ manager_bonus_1 | int }} ₽ фикс.
  • Тип 2: +{{ manager_bonus_2 | int }} ₽ фикс.
  • Тип 3: +{{ manager_bonus_3_per_ticket | int }} ₽ × кол-во любых тикетов (из раздела "Оспаривания")
